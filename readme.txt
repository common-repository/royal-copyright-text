This plugin add copyright text on copy.


== Description ==

This plugin add copyright text after selected content in clipboard. So when visitor copy & past content form your website automatically add this page permalink & copyright notice.


See a full description here:

http://royaltechbd.com/royal-copyright-text

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `royal-copyright-text` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add enjoy
